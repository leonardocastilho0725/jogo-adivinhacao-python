# 🎲 Jogo de Adivinhação em Python

Um projeto divertido para praticar lógica de programação, onde o usuário tenta adivinhar o número secreto gerado pelo computador.

## 🛠️ Funcionalidades
- **Geração de Número Aleatório:** O computador escolhe um número secreto.
- **Sistema de Palpites:** O jogador recebe dicas se o número é maior ou menor.
- **Contagem de Tentativas:** Mostra quantas vezes o jogador tentou até acertar.

## 🚀 Como Jogar
1. Clone o repositório ou baixe o arquivo `.py`.
2. Execute o script no terminal:
   ```bash
   python jogo-de-adivinhação.py
   ```
