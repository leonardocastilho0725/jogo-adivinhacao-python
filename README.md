# 🎮 Jogo de Adivinhação em Python

Este é um dos meus primeiros projetos estudando Python! É um jogo simples de console onde o usuário deve adivinhar o número secreto gerado pelo computador.

## 🚀 Como o jogo funciona
O programa sorteia um número aleatório entre 1 e 100 usando a biblioteca random. O jogador insere palpites através do terminal. O programa dá dicas se o número secreto é maior ou menor que o palpite atual.

## 🧠 Conceitos Praticados
* **Funções (def)**: Organização do código.
* **Laços de Repetição (while)**: Para manter o jogo rodando.
* **Condicionais (if/else)**: Para validar os palpites.
